vetores detectados

O QUE VERIFICAMOS.

CORS INSEGURO
Access-Control-Allow-Origin com wildcard ou origens não autorizadas expondo dados cross-origin.
SECURITY HEADERS
X-Frame-Options, CSP, HSTS, X-Content-Type-Options e Referrer-Policy ausentes ou mal configurados.
SSL / TLS
Versão do protocolo, validade do certificado, cipher suites e redirecionamento HTTPS forçado.
RATE LIMITING
Ausência de throttling expõe a API a força bruta, credential stuffing e ataques de enumeração.
INJEÇÃO SQL / XSS
Parâmetros sem sanitização que permitem injeção de código malicioso nas respostas da API.
AUTENTICAÇÃO
Endpoints sem auth, tokens JWT fracos, sessões sem expiração e configurações de acesso inseguras.
INFORMATION DISCLOSURE
Stack traces, versões de software e dados internos expostos involuntariamente nas respostas HTTP.
OPEN REDIRECT
Redirecionamentos não validados para destinos externos usados como vetor em ataques de phishing.
SSRF
Server-Side Request Forgery — exploração do servidor para acessar recursos internos não autorizados.
